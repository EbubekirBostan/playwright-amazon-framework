# Playwright Amazon Enterprise Framework

Enterprise-style Playwright + TypeScript automation framework.

## Features

- Playwright Test Runner
- TypeScript
- Page Object Model (POM)
- Smoke & Regression tags
- Global Setup (storage state)
- Consent handling for public sites
- Allure Reporting
- Multi-browser support

## Installation

npm install
npx playwright install

## Run Tests

npx playwright test --project=chromium
